# منصة تحليل الاختبارات ونتائج الطلاب

أداة تفاعلية تعمل في المتصفح لتحليل درجات الطلاب إحصائيًا: المتوسطات والوسيط والمنوال، الرسوم البيانية، توزيع التقديرات، تحليل الفقرات (الصعوبة والتمييز)، مؤشر التضخم في الدرجات، مع دعم رفع ملفات Excel/CSV والطباعة والتصدير.

## الاستخدام

افتح `index.html` في أي متصفح حديث، أو استخدم رابط GitHub Pages.
